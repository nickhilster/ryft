# Ryft Visual Rebrand Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the "Voltage Instrument Deck" ryfine visual identity with a clean, corporate-blue Ryft brand across the web app.

**Architecture:** CSS custom property token swap (lime → blue palette, drop Bricolage Grotesque display font), simplified App.css background effects, string replacements throughout App.tsx and components, and workspace package rename from `@ryfine/core` → `@ryft/core`.

**Tech Stack:** CSS custom properties, React/TSX string edits, npm workspace package.json renames.
